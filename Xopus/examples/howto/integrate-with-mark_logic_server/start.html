Mark Logic Server integration example

This example demonstrates a typical Content Management setup with Xopus on Mark Logic Server.

Installation

This demo directory must be placed in the root document directory of a Mark Logic App Server (HTTP). We're assuming you're running the demo on a test server, with access to an administrative account. To run the demo, you must have the demo documents installed in Mark Logic. Run the actions/install.xqy script to remove any existing demo files and install the demo documents. We advise to use a separate document database for this demo.

Download Xopus separately from xopus.com/download.html and unpack the archive in the root document directory.
