Content Management Example

This example is an overview of what to pay attention to when using Xopus in a CMS. It details the following things:

  • Installing Xopus
  • Give Xopus a place in the CMS
  • Creating an Island
  • Communicating with the editor
  • Loading XML
  • Saving
  • Using the API
  • More configuration
  • XSL possibilities

This example is complex. It covers a lot of different topics. Each of these topics is handled in a document in the CMS which refers to the files of the example itself.

This example was written for C# .Net. It will not work properly on servers that do no support this framework.
